# Budget_tracker

A simple console-based budget tracker written in C#.  
This program helps you manage your money by tracking income, expenses, and providing summaries.

## Features
- **Balance Tracking**: Keeps track of your current balance as you add income or expenses.
- **Expense Recording**: Log expenses with store name, category, and amount.
- **Income Recording**: Add income sources, always stored as positive amounts.
- **Transaction History**: View a list of all past transactions with date, store, category, and type.
- **Weekly Summary**: See total income, total expenses, net change, and expense breakdown by category for the last 7 days.
